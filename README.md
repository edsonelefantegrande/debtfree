# 💸 DebtFree - Calculadora de Dívidas

**DebtFree** é uma calculadora simples e open source para ajudar pessoas a organizarem suas dívidas.  
Com ela, você pode simular quanto tempo levará para quitar uma dívida com base no valor inicial, taxa de juros e pagamento mensal.

🔗 **Acesse online aqui:** [https://seu-usuario.github.io/debtfree/](https://seu-usuario.github.io/debtfree/)

---

## 🚀 Funcionalidades
- Inserir valor da dívida, juros ao mês e pagamento mensal.
- Calcular o tempo necessário para quitar a dívida.
- Mostrar aviso se o pagamento não for suficiente para cobrir os juros.

---

## 📂 Como usar localmente
1. Baixe ou clone este repositório:
   ```bash
   git clone https://github.com/seu-usuario/debtfree.git
````

2. Abra o arquivo `index.html` em qualquer navegador.
3. Preencha os campos e clique em **Calcular**.
